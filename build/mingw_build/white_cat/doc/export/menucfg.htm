L'accès à la fenêtre de configuration

A partir de la version 0.7.6, l'accès au menus DMX / Midi / Network et artnet affect change.

Tout se fait depuis une même fenêtre, qui permet un accès aux différents paramètres configurables.

Pour y accéder, clicker CFG MENU ou taper [SHIFT][F11].

Dans cette fenêtre les menus permettent d'accéder aux options diverses. Choisissez l'onglet que vous désirez utiliser.
