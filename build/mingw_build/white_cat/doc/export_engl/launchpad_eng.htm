Here are the assigned colors in White Cat:

Faders

Up, Down, Saw, Flash: orange (col 1)

Lock: green (col 2)

Prev/NextDock, Updown: amber (col 5)

StopPos: red (col 4)

Loop One, Loop All: dark orange (col 6)

Lock Presets: green (col 2)

Play, Seek, Loop Chaser embeded: orange ( col 1)

CueList

Go or pause: red (col 6)

Go Back: red (col 6)

Slaviness of GridPlayer1 accelerometer to CueList accelerometer: orange (col1)

AudioPlayers

Play, General Loop: orange (col 1)

CueLoop: red (col 4)

Banger

Bang It !: red (col 4)

Big commands

Store, Modify,… → AudioPlayers: red (col 4)

Time

In, Out, DIn, Dout, Affect: dark orange (col 6)

Trichromy

Dock color selected: dark orange (col 6)

VideoTracking

Roi selected: dark orange (col 6)

Preset selected: yellow (col 3)

Chasers

Play, Loop, Forward, Backward, Go Back: orange ( col 1)

Time Mode: orange ( col 1)

On/Off tracks: green ( col 2)

FadeIn: orange ( col 1)

Stay: green( col 2)

FadeOut: red ( col 4)

Steps that are being assigned by the operator take the color of the operation (green for up action, orange for stay state, red for down action).

iCat

Selected Preset: dark orange (col 6)

MidiMute (Global)

MidiMute (Global): red ( col 4)

Windows calls

All windows states are in orange ( col1) except:

  • Direct CH: dark orange (col 6)
  • VIEW: dark orange (col 6)
  • BLIND: red ( col 4 )

GridPlayers

On/Off show GridPlayer: red ( col 4 )

Play: orange ( col 1 )

AutoStop Mode: red ( col 4 )

Slave Mode: dark orange ( col 6)

On Off of the 4 macros ( Goto/Count/Seek/StopPlay): red ( col 4 )

Table of values of Launchpad

13 Red Low
15 Red Full

60 Green Full

In Flash mode:
