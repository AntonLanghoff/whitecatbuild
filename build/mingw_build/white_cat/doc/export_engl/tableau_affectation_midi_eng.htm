MIDI table of assignments

Up until version 0.7.6.6 the type of midi signal (Key-On, Key-Off or Ctrl-Change) and from which midi page it was assigned (Faders / Button) was the system in use. From version 0.7.6.6 onward every signal type (Note, Key-On, Key-Off, Ctrl-Change) may be assigned to any command, from only one page: the midi assignment page.

Comparing command ID's is now done only on Midi Channel and Pitch. Only the Flash Key needs to have a signal type of Key-On.

This table presents all commands that can be controlled from midi in White Cat. In physical commands, the recommended one appears first: if it is a knob rather a slider, Knob will be written first, then Slider.

From version 0.8.2.2 onward, auto de-assginment no longer reads the type and focuses only on channel IDs and Pitch.
