FAQ

INSTALLATION

White Cat comes in a ZIP file. There is no installer, so all you need to do is unzip it, the best place being in the root directory (C:/ D:/ etc….)

I'm running White Cat and it says "couldn't load fonts/doom.ttf"
  • You are trying to run White Cat directly from the zip. You need to first extract all files to a folder, and then run White Cat from that folder.
I'm running White Cat, but the program stays blocked or doesn't run at all
  • White Cat embeds a Server to receive the art-net signal. This server is opened on startup. It may be blocked by the Windows firewall or by an added firewall (Zone Alarm, etc). You need to either configure it to allow White Cat or you need to disable the firewall.
  • Under Windows7 or Vista, you need to set administrator rights to White Cat (so that it can write and change files). To do this, right click on Whitecat_x_x.exe and go to the advanced tab: check Run as administrator. On certain computers you will need to set the XP Compatibility from SP3 On.
My license doesn't seem to activate
  • Select the MAC address you have sent to register (CFG MENU > NETWORK)
  • Press F5 and enter the license key in the NAME window
  • Re select the CFG window (an orange outline appears when the window is selected)
  • Click the blinking space, to assign the key

If it still doesn't work (the blinking is not stopping), you can manually edit your license outside White Cat software:

  • Close White Cat. Open with notepad the file “license_key.txt” in the folder whitecat/user/.
  • Clear the second line and copy/paste the license key (not the MAC address !!!). Do not add any return or end of line
  • Save and close the notepad, launch White Cat

If the problem still persists, and you are sure that it is a good MAC Address that was sent to register, please post on forum.

USING WHITE CAT

When I set a channel to be at Full, it shows a different value ( 32%, 86, ... anything other than 100%)
  • In CueList [F9] the Stage master is not at FULL
When I try to manipulate channels with the Up or Down buttons or with the mouse wheel, nothing happens
  • have a look in CueList [F9] to see if the Stage Master is at 0%.
  • check that you are not in [Blind] mode [Shift-F11] setting the channel levels of the memory in preset
  • if not any of these, go to the [CFG Menu] > General tab and check that the default step level is not at 0. This might occur if loading a show from a very very old version of White Cat (previous to 0.7.6.2)
I can't move the Grand Master slider
  • by default this is blocked, especially for usage with video in VVVV, where certain channels are presets of positions (non-intensity parameters), like on moving heads (intelligent lights). To unlock the grand master > CFG MENU > Général
I'm using faders to create memories, but I don't understand how to navigate inside memories
  • The heart of White Cat is the CueList, where memories are recorded. It is by navigating inside the CueList that you can edit the memories. Assigning a memory to a fader's dock is more of an added plus than the main goal in White Cat. White Cat is not a Zéro88
I have shows in the saves folder that are copy-protected that I can't delete or move, which are doing 0Ko weight or have files of 0ko inside

This a bug that was resolved in version 0.7.6.4.

To get rid of those files:

  1. download and install the freeware unlocker. http://ccollomb.free.fr/unlocker/
  2. restart Windows. Give administrator rights to unlocker if you are using Vista or Windows7.
  3. right-click the folder → unlocker → delete.
I m trying to launch a banger and nothing happens

Please verify that in [CFG Menu] > Core Banger is ON.

Where can I directly manipulate channels in midi

There is no other way than using Faders in Direct Channel mode. You will need to assign the desired channel to a fader, then assign this fader the midi signal.
