Enregistrement de WhiteCat et mailing utilisateurs

Licensing or registering

Until version 0.8.3.1, you need a license key ( freely generated) to be able to save your shows. With version 0.8.4, WhiteCat is now opensource and can be used on any computers in a free and not restrictive manner.

In need of a license key for previous versions

If you need to use a previous version and obtain a license key:

  1. open whitecat/user/
  2. edit with the notepad license_key.txt
  3. replace the second line by whitecat magic password
  4. save the text file and run whitecat
