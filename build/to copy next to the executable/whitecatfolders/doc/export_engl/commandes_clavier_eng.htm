Keyboard shortcuts

This assumes you are using an English keyboard. If so, the second line down will read Qwerty. If it reads Azerty, you are on a French keyboard. On the French keyboard, the A & Q keys are reversed as are the W & Z keys. The French M key is the English colon key. The French colon key is the English period dot key, and the French period dot key is the English M key. While the shortcuts below have been footnoted for the English keyboard, the actual typing of names still requires that you remember what keys have been substituted for what others because when you type “mynewshow” it will type ”.YNEZSHOZ”

Global functions

Keyboard Syntax
Quick save [CTRL][S]
Quit and save [CTRL][F12] (Works only in registered version)
Quit without save [SHIFT][F12]
Snapshot and closing windows [SHIFT][PRINT SCREEN]
Recall windows snapshot [CTRL][PRINT SCREEN]
Cycle thru open windows [PGUP] and [PGDOWN]
Open windows in solo mode [CTRL][PGUP] and [CTRL][PGDOWN]

Modify mode [F2]
Report mode [F3]
Clear Mode [F4]
Name [F5]
Time [F6]
Trichromy [F7]
Video Tracking [F8]
CueList [F9]
Faders [F10]
MiniFaders [Shift-F10]
Banger [F11]
Blind mode [Ctrl-F11]
CFG Menu [Shift-F11]
Chasers [Shift-C]
Mover [Shift-M]*

* [Shift-;] on an English Keyboard

Manipulating channels

Selection

Keyboard Syntax
add channel to selection [+] [ch][+]
delete channel from selection [-] [ch][-]
thru selection [TAB] [1st ch][+] [last ch][TAB]
all channels [Y]
unselected channels [U]
clear selection [ESC]

moving selection

left [←]
right [→]

Giving level to the selection

Keyboard Syntax
FULL [I]
At zero [O]
giving a level [ENTER][level][ENTER]
plus %[ARROW UP]
minus %[ARROW DOWN]

Check Channel

Keyboard Syntax
Check ch - [CTRL][←]
Check ch + [CTRL][→]

Copy Paste and Get

Keyboard Syntax
Copy to the Clipboard [CTRL][C] Select channels then [CTRL][C]
Paste [CTRL][V]
Get Function [CTRL][G] Select channels, enter memory nr and [CTRL][G]

Memory recording/deleting

Keyboard Syntax
Create or over record a memory number [SHIFT][F1] [memory nr][SHIFT][F1]
Create memory adding the faders [SHIFT][F3] [memory nr][SHIFT][F3]
Delete memory [SHIFT][DEL] [memory nr][SHIFT][DEL]
Over record on stage or blind [CTRL][F1]
Over record on stage or blind with faders [CTRL][F3]
Reload a memory [CTRL][Z]*
Reload a deleted mem [SHIFT][Z]* [memory nr][SHIFT][Z]*
GET level of selected channels from memory nr [CTRL][G] [mem][CTRL][G]

* [CTRL][W] and [mem][SHIFT][W] on an English keyboard

Navigating thru memories

keyboard Syntax
Previous mem onstage [CTRL][W]*
Next mem onstage [CTRL][X]
Previous mem on preset [SHIFT][W]*
Next mem on preset [SHIFT][X]

* [CTRL][Z] and [SHIFT][Z] on an English Keyboard

Crossfade

Keyboard Syntax
GO/PAUSE [SPACE]
GO BACK [CTRL][SPACE]
DOUBLE GO [SHIFT][SPACE]

Time syntax

Keyboard Syntax
minutes [.][.] [num][.][.]
seconds [.] [num][.]
1/10th of seconds [.][num]

Patch

Keyboard Syntax
open Patch [SHIFT][P]

Dimmers selection

Keyboard Syntax
add to selection [SHIFT][+] [SHIFT][dim][+]
delete from selection [SHIFT][-] [SHIFT][dim][-]
thru selection [SHIFT][TAB] [1st dim][SHIFT][+] [last dim][SHIFT][TAB]
select all dimmers [SHIFT][Y]
all unselected [SHIFT][U]
clear selection [SHIFT][ESC]

Assign a dimmer to a channel

keyboard Syntax
assign dimmer(s) to selected channel [SHIFT][ENTER] [ch][+][dim][SHIFT][+][SHIFT][ENTER]
or [dim][SHIFT][+][ch][+][SHIFT][ENTER]

VideoTracking

Keyboard Syntax
Select Presets 1 to 6 [W][X][C][V][B][N]*
