Core Configuration

(0.8.2.3)

White Cat is a quickly changing project.

It is much more than just a single Cuelist, and there are a lot of background calculations to enable the lighting effects:

  • 1 * 512 channel buffers for Cuelist
  • 48 * 7 512 channel buffers for faders
  • 1 512 channel buffer for video tracking
  • 8 buffers of 512 channels for trichromy
  • 128 * 24 512 channel buffers for chasers
  • 4 buffers 512 channel buffer for gridplayers

The Core tab enables you to lighten White Cat's calculations in the background, especially on small capacity computers.

You can customize to your needs the general options, and avoid calculations if you never use them:

Max. Calculations

This where you define, when to set On/, maximum of:

  • Faders (48)
  • Docks per fader (6)
  • Chasers (127)
  • Tracks per chaser (24)
  • GridPlayers (4)
  • Bangers (127)

Do Calculations

Enable/disable calculations for:

  • faders
  • cuelist
  • bangers
  • trichromy
  • video-tracking
  • chasers
  • gridplayers

Set to Off, buffers and depending buffers that are no longer being calculated (LFO, cross-fade, chaser time, etc…) and which will stay frozen.

All those options are saved in the folder /user file config_core.txt. You may edit it manually as a last resort.

Be aware, if you turn off the Cuelist, this will block the Direct Channel mode of the Faders.
