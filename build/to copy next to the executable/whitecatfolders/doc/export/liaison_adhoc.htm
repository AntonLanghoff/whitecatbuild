Liaison ad-hoc

Une liaison ad-hoc permet de connecter jusqu'à 9 machines entre elles, en utilisant la carte wifi de votre machine.

Si vous utilisez une interface DMX art-net en RJ45, il faudra rajouter un routeur wifi à 25 euros, pour partager la connexion à la carte réseau de whitecat. ( Pour l'instant WhiteCat se connecte à une seule carte réseau).

Créer une liaison ad-hoc

XP

Les procédures diffèrent entre XP et Seven. Sous XP la configuration ad-hoc est un peu plus laborieuse que sous Seven.

  • ouvrir panneau configuration > connexions reseaux > Connexions réseaux sans fils > Propriétés
  • clicker l' onglet Configuration réseaux sans fil > avancé > réseau d'égal à égal (ad-hoc)
  • sélectionner: ne pas connecter automatiquement
  • aller dans > ajouter > donner un nom
  • paramétrer en mode ouvert sans clé wep > clicker ok
  • rafraichir les connexions réseaux sans fil pour afficher votre réseau ad-hoc, le sélectionner et connecter dans windows
  • dans l'iPod/Phone: rafraichir les reseaux et se connecter au réseau ad-hoc.

Si vous utilisez votre machine pour aller sur internet en tournée, il faudra reconfigurer votre carte réseau en mode Tout point d'accès ( Configuration réseaux sans fil > avancé > ). Certains points d'accès wifi demandent aussi que l'adresse IP soit automatiquement attribuée, et pas fixe.

VISTA

  • Cliquer sur le bouton Démarrer.
  • Cliquer sur Connexion.
  • Cliquer sur Configurer une connexion ou un réseau.
  • Cliquer sur Configurer un réseau ad hoc (ordinateur à ordinateur) et Suivant.
  • Entrer le Nom du réseau, par exemple “iCat connexion”.
  • Cocher la case Enregistrer ce réseau.

SEVEN

  • Cliquer sur le bouton Démarrer.
  • Cliquer sur Panneau de configuration.
  • Cliquer sur Réseau et Internet.
  • Cliquer Centre de réseau et partage.
  • Cliquer sur Configurer un Réseau sans fil ad hoc (ordinateur à ordinateur).
  • Cliquer deux fois sur Suivant.
  • Entrer le Nom du réseau, par exemple “iCat seven”.
  • Cliquer sur Partager la connexion Internet.
